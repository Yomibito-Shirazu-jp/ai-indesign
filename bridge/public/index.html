AI in Design 接続サポート

InDesignとブリッジサーバーの通信状況を確認するためのツールです。AI未契約でも利用できます。

1. InDesign通信ステータス

サーバーと通信中...

うまく接続できない場合：

  1. InDesignを起動してください。
  2. UXP Developer Tool を利用して、「AI in Design」プラグイン（plugin/manifest.json）をロードしてください。
  3. 「Load」をクリックし、パネルを表示させた状態で「更新」ボタンを押してください。

▶ InDesign動作テスト (AI不要)

下記のボタンを押すと、InDesign側でアラートダイアログが表示されます。

2. AI有料契約をしていない方へ (無料の利用方法)

「Claude Proなどの有料AIを契約していないと使えないの？」
いいえ、安心してください！無料のツールやサービスを組み合わせることで、完全無料でAI in Designを利用（またはテスト）することが可能です。

以下のいずれかの方法を利用すれば、有償のAIサブスクリプションがなくても動作させることができます。

  • 方法 A: 【一番おすすめ】VS Code + Roo Code (Cline) + 無料API 無料で使えるエディタ「VS Code」に、「Roo Code (旧Cline)」という無料拡張機能を入れます。
    Gemini 1.5 Pro/2.0 API（無料枠あり）などをAPIキーで連携し、MCPサーバーとして「AI in Design」を設定すれば、チャット形式でInDesignを操作できます。
  • 方法 B: Cursorエディタ (無料体験) 人気のAIエディタ「Cursor」（基本無料）をダウンロードします。CursorのMCP設定画面から、このシステムの `npm start` を登録するだけで、エディタ内のチャットからそのままInDesignに指示が出せます。
  • 方法 C: Ollama + Chatbox (完全ローカル・無料) Ollamaなどを利用してPC内で無料AI（Llama3やQwenなど）を動かし、「Chatbox」などのMCP対応GUIアプリに連携させます。完全にオフライン＆無料で操作が可能です。

※ 「Claude Desktop」にこだわらなくても、近年様々な無料の「MCP対応クライアント」が登場しています。
「MCP クライアント 無料」などで検索し、ご自身にあったツールをお選びください。
