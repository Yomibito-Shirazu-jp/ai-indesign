ai-indesign — 本サーバー（このファイル）
1
Node.js — サーバー起動確認
このページが表示されていれば完了
Bridge serverが正常に起動しています。Node.js環境は問題ありません。
2
InDesign UXP Plugin をインストール
.ccxファイルをダウンロードしてダブルクリック
Ai-inDesign.ccx をダウンロード
  1. 上のボタンで Ai-inDesign.ccx をダウンロード
  2. ダウンロードした .ccx ファイルをダブルクリック
  3. Adobe Creative Cloud がインストールを自動実行
  4. InDesign を起動（または再起動）
  5. メニュー 「ウィンドウ」→「プラグイン」→「Bridge Panel」
CCXが動かない場合はUXP Developer Toolで plugin/ フォルダを直接ロードしてください。
3
Claude Desktop に接続設定を追加
claude_desktop_config.json にMCPサーバーを登録

以下のJSONを %APPDATA%\Claude\claude_desktop_config.json に追加してください。
Configタブの「Claude Desktop Config」からもコピーできます。 

{
  "mcpServers": {
    "ai-indesign": {
      "command": "node",
      "args": ["C:\\path\\to\\ai-indesign\\src\\index.js"]
    }
  }
}
設定後は Claude Desktop を完全に終了してから再起動してください（タスクトレイ→右クリック→Quit）。
4
接続テスト
全システムの通信を確認する

Dashboardのステータスで「Bridge Server」と「InDesign Plugin」が両方オンラインになっていれば成功です。
