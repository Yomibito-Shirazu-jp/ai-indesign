📋 インストール手順：

  1. 上のボタンで Ai-inDesign.ccx をダウンロード
  2. ダウンロードした .ccx ファイルをダブルクリック
  3. Adobe Creative Cloud が自動的にインストールを実行します
  4. InDesign を起動（または再起動）
  5. メニュー 「ウィンドウ」→「プラグイン」→「Bridge Panel」 をクリック
⚠️ CCXインストールが動かない場合は、UXP Developer Tool を使って plugin/ フォルダを直接ロードすることもできます。詳しくはFAQを確認してください。
3
Claude Desktop に接続設定を追加
バッチファイルで自動設定済みのはずです

起動バッチファイル（AutoDTP_起動.bat）を実行済みなら、Claude Desktop の設定は自動で完了しています。

⚠️ 重要: Claude Desktop を一度完全に終了（タスクトレイ→右クリック→Quit）してから再起動してください。設定は再起動しないと反映されません。

手動で設定する場合は、以下のJSONを claude_desktop_config.json に追加してください：

{
  "mcpServers": {
    "ai-indesign": {
      "command": "node",
      "args": ["C:\\path\\to\\src\\index.js"]
    }
  }
}

設定ファイルの場所: %APPDATA%\Claude\claude_desktop_config.json

以下を順番に確認してください：

  2. InDesign のメニュー「ウィンドウ」→「プラグイン」→「Bridge Panel」がクリックされているか？

🤖 Claude Desktop で「ai-indesign」が表示されません ▼
  1. Claude Desktop を完全に終了してください（タスクトレイ→右クリック→Quit）
  2. Claude Desktop を再起動してください
  3. 設定画面の「Developer」→「MCP Servers」に「ai-indesign」が表示されるか確認

表示されない場合は、AutoDTP_起動.bat を再実行して設定を自動追加してください。

⚡ 「ポート3000が使用中」と表示されます ▼

既にブリッジサーバーが起動している可能性があります。

  • 既に開いているコマンドプロンプトのウィンドウがないか確認
  • タスクマネージャーで「node.exe」プロセスを終了
  • PCを再起動してからバッチファイルを再実行
💻 Node.js がないと言われます ▼

https://nodejs.org/ja/ からLTS版をダウンロードしてインストールしてください。

インストール時は、すべてデフォルト設定のままで「Next」をクリックし続けるだけでOKです。インストール後、PCの再起動が必要な場合があります。

🔑 Claude Desktop は有料ですか？無料で使えますか？ ▼

Claude Desktop 自体は無料で利用できます（基本のChatモード）。MCP機能も無料プランで使えます。

より高度な利用をする場合は、Proプラン（月額$20）でより長い会話が可能です。

Claude Desktop 以外にも、以下の無料ツールからMCP接続できます：

  • VS Code + Roo Code — 無料のAIコーディングツール
  • Cursor — 無料体験あり
🔒 セキュリティ上の懸念はありますか？ ▼

ブリッジサーバーは localhost（127.0.0.1） のみでリッスンしています。外部からのアクセスは受け付けません。

通信はすべてお使いのPC内で完結するため、社外にデータが流出することはありません。AIサービス（Claude/Gemini等）を使う場合のみ、テキストデータに限りクラウドに送信されます。
